== Description ==
This plugin adds an Image + Text Carousel widget for Elementor, allowing users to design custom sliders with images, text, links, and styling options.

== Installation ==
1. Upload the plugin ZIP via Plugins → Add New
2. Activate the plugin
3. Open Elementor Editor → Search “Image Text Carousel”
